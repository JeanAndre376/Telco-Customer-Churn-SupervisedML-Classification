# Telco-Customer-Churn-SupervisedML-Classification
Churn prediction using the IBM Watson Telco dataset (Kaggle). 7,043 customer records, 20 features, binary classification target. Models compared: Logistic Regression, Decision Tree, Random Forest &amp; XGBoost. Best model selected via AUC-ROC, Recall, and F1 Score.
